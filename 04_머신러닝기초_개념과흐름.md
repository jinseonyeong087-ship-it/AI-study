# 4. 머신러닝 기초: 개념과 흐름

## 머신러닝 한 줄 정의
데이터를 통해 규칙을 학습해서 예측/분류를 수행하는 방법.

## 지도학습/비지도학습
- 지도학습: 정답 라벨 있음 (정상/불량)
- 비지도학습: 정답 라벨 없음 (군집화 등)

## 기본 파이프라인
1. 데이터 수집
2. 전처리
3. 학습/검증/테스트 분리
4. 모델 학습
5. 성능평가
6. 개선 반복

## 주요 알고리즘(입문)
- 로지스틱 회귀
- SVM
- 랜덤포레스트

## 평가 지표
- 정확도(Accuracy)
- 정밀도(Precision)
- 재현율(Recall)
- F1-score

불량 검출에서는 보통 **재현율(불량 놓치지 않기)** 이 더 중요할 때가 많음.
