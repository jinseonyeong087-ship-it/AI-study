# 2. 개발 환경 세팅 (윈도우 기준)

## 권장 구성
- Python 3.10~3.11
- VS Code
- Anaconda 또는 venv
- Git
- (선택) NVIDIA GPU + CUDA

## 최소 설치 순서
1. Python 설치
2. VS Code 설치
3. 가상환경 생성
4. 필수 패키지 설치

```bash
python -m venv .venv
.venv\Scripts\activate
pip install --upgrade pip
pip install numpy pandas matplotlib scikit-learn jupyter opencv-python ultralytics
```

## 확인 테스트
```bash
python -c "import cv2, sklearn, torch; print('ok')"
```

에러가 나면 패키지 버전 충돌 가능성이 큼 → 새 가상환경에서 다시 설치.
