# 6. 딥러닝 기초 (CNN 중심)

## 딥러닝이 필요한 이유
이미지 특징을 사람이 직접 설계하지 않아도 모델이 자동으로 학습.

## 핵심 개념
- 신경망(Neural Network)
- 활성화 함수(ReLU 등)
- 손실함수(Loss)
- 옵티마이저(Adam, SGD)
- 에폭(Epoch), 배치(Batch)

## CNN 핵심
- Convolution: 특징 추출
- Pooling: 크기 축소 + 요약
- Fully Connected: 최종 분류

## 과적합 방지
- 데이터 증강
- Dropout
- Early Stopping
