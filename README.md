## 추천 학습 방법

- 하루 1~2시간 기준으로, **읽기(30%) + 실습(70%)** 비중 추천
- 모르는 용어는 바로 넘기지 말고, 파일 안의 "핵심 용어" 섹션 먼저 확인
- 실습 코드는 직접 타이핑해보기 (복붙만 하면 기억이 잘 안 남음)

## 최종 목표

이 로드맵을 완료하면 아래를 스스로 설명/수행할 수 있습니다.

- 머신러닝/딥러닝의 차이와 불량검출에서의 역할 설명
- 이미지 데이터셋 준비 및 기본 전처리
- YOLO로 기본 학습/추론 실행
- 성능지표(정밀도, 재현율, mAP) 해석
- 현업 투입 전 "무엇을 먼저 확인해야 하는지" 체크 가능
